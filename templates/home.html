🚀 Getting Started (Shuru Kaise Karein)

  1. Get API Token: https://huggingface.co/settings/tokens par jaao
  2. Create New Token: "New Token" button click karo
  3. Copy Token: Token copy karo (hf_ se start hota hai)
  4. Paste Here: "ACCESS TOKEN" panel mein paste karo
  5. Click SAVE: Save button click karo
  6. Upload Image: Koi bhi image upload karo
  7. Click INITIALIZE: "INITIALIZE" button ko click karo
  8. Launch Analysis: Popup mein "LAUNCH ANALYSIS" click karo

⚠️ Common Errors (Agar Error Aaye)

❌ "Token must start with hf_"

Matlab: Token galat format mein hai

Fix: Pura token copy karo jo hf_ se start hota hai. Koi part miss na ho

❌ "Invalid API Token"

Matlab: Token expired hai ya galat hai

Fix:
→ https://huggingface.co/settings/tokens par jaao
→ Purana token delete karo
→ Naya token create karo
→ Naya token paste karke save karo

❌ "Service Overloaded" ya "Model Unavailable"

Matlab: Hugging Face ka server busy hai

Fix:
→ 30 second wait karo
→ Page refresh karo (F5)
→ Dobara try karo

❌ "Invalid Image Format"

Matlab: Image format galat hai ya bahut bada hai

Fix:
→ PNG, JPG, ya WebP format use karo
→ Image ka size 5MB se kam hona chahiye
→ Dobara try karo

❌ "Network Error" ya "Failed to fetch"

Matlab: Internet connection issue hai

Fix:
→ Internet connection check karo
→ Agar VPN use kar rahe ho to band karo
→ Page refresh karo
→ Dobara try karo

❌ "CORS Error"

Matlab: Browser cross-origin request block kar raha hai

Fix:
→ Page ko hard refresh karo (Ctrl+Shift+R)
→ Browser cache clear karo
→ Dobara try karo

💡 Troubleshooting Tips

✅ Pehle Yeh Check Karo:

  • Internet connection is working
  • Token starts with "hf_" (copy-paste properly)
  • Image format PNG/JPG/WebP hai
  • Image size 5MB se kam hai
  • Browser developer console (F12) mein koi error dikhe to report karo

🔍 Browser Console Dekhne Ke Liye:

  1. F12 press karo
  2. "Console" tab click karo
  3. Red error messages dekho
  4. Iss error ko apne help kar sakte ho par detailed information milti hai

🔐 Token Security:

  • Token browser ke localStorage mein save hota hai (sirf tumhara device)
  • Kisi ko apna token share mat karo❌
  • Token kisi bhi server par nahi bheja jata ✅
  • Share karne se pehle private browser tab mein check karo

❓ Frequently Asked Questions (FAQs)

Q: Mera token kahan save hota hai?

A: Browser ke localStorage mein sirf tumhare device par. Kisi server par nahi

Q: Kya free hai ye service?

A: Haan! Hugging Face free hai. Bas account banana padta hai

Q: Multiple images analyze kar sakta hoon?

A: Bilkul! Ek ek karke images upload aur analyze kar sakta hai

Q: Result copy kar ke save kar sakta hoon?

A: Haan! 📋 COPY button se result copy kar sakta hai

Q: Agar fir bhi kaam na kare?

A: Browser refresh karo (F5) aur dobara try karo. Lagbhag sab issues solve ho jate hain

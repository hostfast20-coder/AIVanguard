Style Transfer for Website Design

This feature allows users to upload a screenshot of any website they like, and the system automatically generates Tailwind CSS code that mimics the original design's color scheme, typography, spacing, and layout structure. Using computer vision techniques, the tool analyzes visual elements such as background colors, font styles, button shapes, alignment, and grid patterns. The extracted design information is then processed through an AI model that converts visual patterns into structured Tailwind utility classes. This enables fast design-to-code transformation, helping developers replicate or get inspired by existing designs without manually inspecting CSS. The project integrates image analysis, CSS generation, LLM-based reasoning, and modern web design principles to create an intelligent and efficient styling assistant.
